# **App Name**: Vortex Arena

## Core Features:

- Smart Seeding Assistant: An AI-powered tool that evaluates player historical performance and team stats to suggest optimal tournament brackets and fair seedings.
- Liquid Bracketing Engine: An interactive, horizontal-scrolling knockout bracket visualizer with dynamic curved line connections and glassmorphic node styling.
- Live Pulse Scorekeeper: Mobile-optimized interface for referees with oversized tactile buttons, pulsating 'Live' indicators, and high-visibility glass components for outdoor use.
- Admin Management Command: Full-featured dashboard with fixed glass sidebars and metric cards tracking revenue, active matches, and participant density in real-time.
- Captain’s Team Portal: Mobile-first hub with a bottom glass navigation bar for managing player rosters and viewing centralized team match schedules.
- Fan Experience Hub: A public-facing destination with active tournament grids, immersive hero sections, and real-time score feeds using glassmorphism effects.

## Style Guidelines:

- Primary color: Vibrant Electric Blue (#1E61FF) to evoke high-performance athleticism and precision.
- Background color: Deep Abyss Navy (#05070D), providing a dark, high-contrast canvas for liquid mesh gradients in purple, neon blue, and cyan.
- Accent color: Neon Cyan (#00E5FF) to highlight live status indicators and call-to-action elements.
- Font pairing: 'Space Grotesk' (sans-serif) for high-impact headlines to provide a tech-forward aesthetic, and 'Inter' (sans-serif) for dense data views and readability.
- Glassmorphism system utilizing white-tinted glass (white/5) for cards with multi-layered 'backdrop-blur-xl' and razor-thin white/10 borders.
- Monochrome stroke icons with a slight neon glow effect to match the 'Liquid Glass' theme.
- Framer Motion organic fluid transitions for modals and morphing mesh background gradients that react slowly to user navigation.
